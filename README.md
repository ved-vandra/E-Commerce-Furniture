# 🛋️ E-Commerce Furniture Sales Analysis

## 📌 Project Overview
This project analyzes E-commerce furniture sales data to understand sales performance, customer behavior, and product trends.  
The goal is to find useful insights that help improve business decisions.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Power BI
- Google Colab

---

## 📂 Project Files
- `ECommerce_Furniture.ipynb` → Data analysis using Python  
- `ecommerce_furniture_dataset_2024.csv` → Dataset used in project  
- `E-Commerce-Furniture-Sales-Dashboard.pbix` → Power BI dashboard file  
- `E-Commerce Furniture Document.pdf` → Project documentation  

---

## 📊 Key Analysis Done
- Data cleaning and preprocessing  
- Sales trend analysis  
- Product category performance  
- Customer behavior analysis  
- Revenue insights  

---

## 📈 Dashboard Insights
- Identified top-selling furniture categories  
- Found high revenue-generating products  
- Analyzed sales trends over time  
- Compared performance across categories  

---

## ▶️ How to Run the Project
1. Download the `.ipynb` file  
2. Open it in Google Colab or Jupyter Notebook  
3. Upload the dataset file  
4. Run all cells
